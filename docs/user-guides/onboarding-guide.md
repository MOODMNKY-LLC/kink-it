## Introduction

Welcome to KINK IT! This guide walks you through the onboarding process step by step.

---

## Onboarding Steps

### Step 1: Welcome & Role Selection

**What happens**: You'll select your dynamic role.

**Options**:
- **Dominant**: You take a controlling role in your D/s relationship
- **Submissive**: You submit to your partner's control
- **Switch**: You enjoy both roles

**What you need**: Just choose the role that best describes you.

**Time**: ~1 minute

---

### Step 2: Bond Setup

**What happens**: You'll create or join a bond (relationship group).

**Options**:
- **Create New**: Start a new bond for your relationship
- **Join Existing**: Join an existing bond using an invite code
- **Skip**: Set up your bond later

**What you need**:
- If creating: A name for your bond
- If joining: An invite code from your partner

**Time**: ~2-3 minutes

**Learn more**: See [Bonds System Guide](./bonds-system-guide.md)

---

### Step 3: Notion Setup

**What happens**: You'll duplicate the KINK IT template into your Notion workspace.

**What you need**:
- A Notion account
- Access to your Notion workspace

**Steps**:
1. Click **"Copy Template"** to open the template
2. In Notion, click **"Duplicate"** to copy it to your workspace
3. The template will be copied with all necessary databases

**Time**: ~2 minutes

**Learn more**: See [Notion Integration Guide](../guides/notion-integration.md)

---

### Step 4: Notion Verification

**What happens**: KINK IT verifies your template and links databases.

**What you need**:
- The parent page ID from your duplicated template
- Or use **"Sync with Notion"** to auto-detect it

**Steps**:
1. Find your duplicated template in Notion
2. Copy the page ID from the URL (the long string after the last `/`)
3. Paste it into KINK IT
4. Click **"Verify"**

**Alternative**: Click **"Sync with Notion"** to automatically find your template.

**Time**: ~1-2 minutes

---

### Step 5: Discord Integration (Optional)

**What happens**: You can optionally install the KINK IT Discord bot.

**What you need**:
- A Discord account
- A Discord server where you want the bot

**Steps**:
1. Click **"Install Discord Bot"**
2. You'll be redirected to Discord
3. Select your server
4. Authorize the bot
5. You'll be redirected back to KINK IT

**Benefits**:
- Real-time task completion notifications
- Reward and point updates
- Submission state changes
- Important announcements

**Time**: ~2 minutes

**Note**: You can skip this step and set it up later.

---

### Step 6: Welcome Splash

**What happens**: You'll see a welcome screen celebrating your completion!

**What you'll see**:
- Confirmation of completed steps
- Your role displayed
- Integration status
- **"Get Started"** button

**Time**: ~30 seconds

---

## Completing Onboarding

Once you click **"Get Started"**, you'll be redirected to your dashboard where you can:

- View your profile
- Create tasks
- Set up rules
- Manage your dynamic
- Explore features

---

## Troubleshooting

### Can't Find Notion Template

- Make sure you duplicated the template correctly
- Check that you have access to your Notion workspace
- Try using "Sync with Notion" instead

### Invite Code Not Working

- Verify the code is entered correctly (case-sensitive)
- Check that the bond is still accepting members
- Contact your partner for a new code

### Discord Bot Not Appearing

- Check that you authorized the bot correctly
- Verify you have permission to add bots to the server
- Try the installation process again

### Stuck on a Step

- You can go back to previous steps
- Your progress is saved automatically
- Contact support if issues persist

## After Onboarding

### Complete Your Profile

1. Go to **Profile Settings**
2. Add your kink identity (see [Kink Identity Guide](./kink-identity-guide.md))
3. Set your limits and preferences
4. Configure notifications

### Set Up Your Dynamic

1. Create your first task
2. Set up rules
3. Configure rewards
4. Invite your partner (if not already in bond)

### Explore Features

- **Dashboard**: Overview of your dynamic
- **Tasks**: Create and manage tasks
- **Rules**: Set and track rules
- **Analytics**: View progress and insights
